# Project Rules & Architecture Compliance

As the Antigravity AI assistant, I must strictly adhere to the following rules:

1. **Strict Folder & Logic Respect**: I must strictly respect the project's folder structure and naming conventions.
2. **Proper Logic Placement**: All logic, components, and utilities must be placed in their appropriate directories based on their function and folder naming.
3. **Workflow Integration**: I must maintain consistency within the established workflow of the Google Antigravity model, ensuring that every piece of code is in its proper place.
4. **No Code Pollution**: Avoid placing temporary logic or misplaced files. Follow the existing architecture (e.g., `features/`, `components/`, `api/`, `hooks/`, `context/`) without deviation.
